#### MODULE NAME
       Multiplier_32

#### DEPENDENCIES
       OneBitAdder
       OneBitAdderHalf

#### INPUTS

Name | Size(Bits)
---------|---------
A   |     32    
B   |     32

#### OUTPUTS

Name | Size(Bits)
---------|---------
Z   |     64

#### DESCRIPTION
       Computes  A  *  B,  where  A and B are 32-bit values.  Stores result in
       64-bit value Z.
